# heffer.info

Personal CV/portfolio site for Joe Heffer. Static HTML/CSS/JS, no build step, no frameworks.

- All content lives in `index.html`.
- Styling is in `assets/css/styles.css`; brand colors are the `--brand-1/2/3` variables in `:root`.
- Behavior (theme toggle, mobile nav, scroll reveal/spy) is in `assets/js/main.js`.
- Deployed via GitHub Pages using `CNAME` and `.nojekyll` — don't remove these.
- Run locally with `python3 -m http.server 8000`.
